    B. COURSE OUTLINES

    Revit is a great BIM program for structural engineering, offering building simulations, and publishing clear, easy-to-share documentation. BIMTech provides this training course focuses on the major components of Revit used for structural design.

    Starting with referencing an architectural model, you'll learn how to add foundations, rebar, and framing; set up views and drawings; model slabs and trusses; and build a whole lot more. This course is designed for students who has already taken BIMTech Revit Architecture Course. Either way, this comprehensive course will teach you what you need to know to be dangerous in Revit structure.

    Topics include:
    • Linking architecture
    • Creating levels
    • Creating view templates
    • Creating foundations, with footings, piers, pilasters, and slabs
    • Adding beam systems
    • Creating a slab floor
    • Reinforcing with rebar
    • Adding brace frames
    • Creating stairs and ramps
    • Inserting AutoCAD geometry
    • Creating schedules
    • Plotting and sharing files
